# Pi Agent GUI 更新日志

这个日志只记录 `tools/pi-agent-gui/` 工具本身的变化，避免混入网站面向玩家的 `base/announcements.json`。

## 2026-06-15

- 优化：移动端工作台补齐 VS Code 式分隔条，支持项目区/聊天区、预览区/对话区、对话区/输入区之间按当前内容动态伸缩。
- 修复：运行面板侧栏分隔条不再被标题、按钮或横向预览分隔条的滚动宽度反向推宽，左右拖动方向与实际指针一致。
- 修复：手机竖屏分隔条保留 12px 触摸热区，但只显示 1px 中线，避免中间出现突兀黑色横条。
- 优化：手机端默认压缩项目区和输入区占比，分隔条改为 1px 布局轨道加 12px 触摸热区，给聊天和预览释放更多空间。
- 修复：手机浏览器按当前可见视口计算工作台高度，避免地址栏/工具栏导致页面比屏幕多出约 100px。
- 修复：手机端预览/对话分隔条现在会实际调整预览高度，并扩大触摸命中区以便手指拖动。
- 优化：低高度横屏小屏改为左右分区，避免预览、对话和输入区互相挤压重叠。
- 修复：手机端项目状态文本保持可见，预览标题图标不再换行，拖动预览时会保留对话区可用高度。
- 优化：小屏下压缩配置表单、变更列表和代码预览的行距与内距，让同一屏可见更多文件和代码内容。

## 2026-06-14

- 修复：预览概览标记按全文件范围抽样显示，避免大文件后半段没有高亮。
- 优化：变更列表增加跳转项目文件入口，可以从 Git 变更直接打开并定位对应文件。

## 2026-06-13

- 新增：完成 Pi 双端 GUI 原型，电脑端和手机端浏览器连接同一个 Pi RPC runtime。
- 新增：支持通过 `start-tailscale.bat` 一键以 Tailscale 方式启动，默认可用 `0.0.0.0:3002` 访问，并支持无 token approve 模式。
- 新增：加入运行状态、会话消息、工具活动、发送模式、Abort、Steer、Follow-up 等共享 agent 控制能力。
- 新增：加入模型 Provider 配置界面，可在浏览器中维护 provider、base URL、API key 引用、模型 ID、上下文和推理相关开关。
- 新增：加入项目文件树、项目搜索、Git 变更列表、文件读取、diff 预览和文件保存能力，让手机端不再只是 chat 入口。
- 优化：界面改为中英双语，可在页面内切换主要控件、说明和状态文案。
- 优化：增加固定说明面板，悬浮、点击或聚焦控件时展示当前位置的功能解释。
- 优化：调整为移动端和桌面端自适应工作台，尽量在固定视口中完成项目、变更、预览和对话操作。
- 优化：预览区增加文件类型图标、语法高亮、搜索命中高亮和 VS Code 风格概览条。
- 优化：diff 预览保留变更增删语义，同时显示代码语法高亮。
- 优化：普通文件编辑复用高亮预览层，避免切到独立编辑框时出现缩放跳变和高亮丢失。
- 优化：概览条拆分为代码结构轨和变更/搜索/错误轨；普通文本行、单行和多行变更标记的显示更加统一。
